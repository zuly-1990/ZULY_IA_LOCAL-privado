Transcripción: Flujo de Trabajo Arquitectónico Avanzado (Villa Savoye)

[5:35] Creación de Columnas Circulares: Shift+A -> Mesh -> Circle. En modo edición: A (seleccionar vértices) -> F (Fill) -> 3 (modo caras) -> E (Extrudear).
[6:32] Requisito CAD: Los elementos (muros, columnas, vidrios) deben ser polilíneas cerradas para facilitar la extrusión directa en Blender. Uso de 'Join' o 'Pedit' en CAD.
[7:41] Exportación: Guardar como AutoCAD 2018 DXF.
[8:36] Resolución de Zoom: Si el zoom falla o es errático, usar el punto (.) del teclado numérico para centrar la vista en el objeto seleccionado. Ajustar 'Navigation' en preferencias para un zoom más suave.
[11:00] Organización de Niveles: Importar plantas por separado. Unir segmentos con Ctrl+J. Desplazar niveles en el eje Z (G Z) para apilar la estructura.
[12:10] Fachadas y Cortes: Rotar 90 grados y usar el Snap (Vertex) para alinear las elevaciones con la planta 2D (imanar vértices).
[14:25] Punto de Pivote: Reubicar el origen del objeto (Set Origin -> Origin to Cursor) usando Shift + Click derecho para posicionar el cursor 3D.
[20:15] Modelado de Losas: Seleccionar contorno con Alt + Click (si es polilínea) -> F (Fill) -> E (Extrude).
[28:12] Separación de Objetos: Seleccionar parte del mesh -> L (Linked) -> P (Separate Selection) para crear objetos independientes (columnas, muros).
[38:53] Limpieza de Geometría: Si hay errores en la extrusión, usar 'Merge by Distance' (M o Clic derecho -> Merge) con distancia 0 para eliminar vértices duplicados o superpuestos.
[40:35] Ventanas y Carpintería: Duplicar aristas (Shift+D), extrudir y repetir acciones con Shift+R (Array manual rápido para rejas o divisiones).
[46:50] Selección en X-Ray: Usar Alt+Z para ver y seleccionar caras inferiores o traseras ocultas.
[53:50] Unión Masiva: Ctrl+J para consolidar elementos similares en un solo objeto antes de organizar colecciones.
[57:26] Gestión de Colecciones: Crear carpetas por niveles (Nivel 1, 2, 3) en el Outliner para mantener el orden arquitectónico.
