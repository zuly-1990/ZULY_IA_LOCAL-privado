# scripts_blender/legacy/

Esta carpeta almacena scripts antiguos, experimentales o en desuso.

- Mueve aquí cualquier script que ya no se use activamente pero que quieras conservar como referencia.
- Así mantienes el directorio principal de scripts limpio y enfocado en lo actual.

---

**¿Por qué archivar?**
- Facilita la organización y el aprendizaje incremental.
- Permite revisar la evolución del proyecto y recuperar ideas pasadas si es necesario.
