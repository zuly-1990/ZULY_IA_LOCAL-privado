COMO COMENZAR
================================================================

1. LECTURA RAPIDA (20 minutos):
   cat bitacora/RESUMEN_EXPANSION_V4.md

2. INICIAR WEB UI:
   python web_ui/app.py
   # Luego visita http://localhost:5000

3. USAR DESDE PYTHON:
   from core.agent import Agent
   agent = Agent()
   agent.process_natural_request('crea un cubo')

4. CAMBIAR DE IDIOMA:
   from core.utils.multilanguage import Language
   agent.set_language(Language.ENGLISH)

5. USAR ANIMACIONES:
   from scripts_blender.animation_engine import AnimationBuilder
   builder = AnimationBuilder()
   builder.rotate_object('Cubo', 'Z', 0, 360)
