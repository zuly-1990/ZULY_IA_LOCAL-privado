## Qué es el Núcleo

El núcleo de ZULY es el conjunto de módulos cognitivos que definen los límites, la filosofía y el control del sistema.

**Módulos del Núcleo (CONGELADOS):**

1. **StateAwareness** (Fase 5.14) - Autoconciencia operativa pasiva
2. **StateGuard** (Fase 5.15) - Límites de uso del estado
3. **IntentionBoundary** (Fase 5.16) - Cortafuegos de intención
4. **CommandGate** (Fase 5.17) - Puerta de comandos explícitos
5. **ExecutionShell** (Fase 5.18) - Cápsula de ejecución neutra
6. **PatternMemory** (Fase 5.13) - Memoria de patrones estructurales
7. **V0Validator** (Fase 5.12) - Validación física existencial

**Estos módulos son BASE INMUTABLE.**

### 1. Cambios Funcionales
- Añadir nuevos métodos a clases del núcleo

### 2. Cambios Cognitivos
- Añadir capacidad de decisión
- Añadir interpretación semántica
- Añadir generación de intención
- Añadir aprendizaje automático

### 3. Cambios Arquitectónicos
- Conectar módulos que están separados
- Romper límites establecidos
- Añadir dependencias entre módulos
- Modificar interfaces

### 4. "Mejoras"
- Optimizaciones de rendimiento
- Refactorizaciones "menores"
- Añadir logging "útil"
- Añadir features "pequeñas"

**Un núcleo congelado NO SE MEJORA. Se respeta.**
