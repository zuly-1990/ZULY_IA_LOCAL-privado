## Documentos Fundacionales

### 🪧 TABLA DE NOÉ (NF-01)
**Archivo:** [`TABLA_DE_NOE.md`](TABLA_DE_NOE.md)

**Principio:** Dignidad del No-Saber

**Rango:** SUPRA-ARQUITECTÓNICO

**Resumen:**  
ZULY reconoce y respeta el derecho de la información a no ser comprendida, no ser reducida a utilidad, y no ser forzada a generar acción. Algunas cosas existen solo para haber sido dichas.

**Estado:** VIGENTE — NO DEROGABLE

*"No todo lo que entra existe para ser usado"*
